# Potencial-Yukawa
Se obtendrán las combinaciones de parámetros libres m, g y λ del modelo de Yukawa. El código de Python siguiente es para comprobar los datos teóricos de Lattice QCD vs los obtenidos mediante el código anterior. Aquí se pueden sustituir cualquier combinación de parámetros iniciales y mostrará la gráfica visual de Lattice QCD
