# Potencial-Yukawa
Se obtendrán las combinaciones de parámetros libres m, g y λ del modelo de Yukawa. El código de Python siguiente es para comprobar los datos teóricos de Lattice QCD vs los obtenidos mediante el código anterior. Aquí se pueden sustituir cualquier combinación de parámetros iniciales y mostrará la gráfica visual de Lattice QCD 

Se obtendrán las combinaciones de parámetros libres $m$, $g$ y $\lambda$ del modelo de Yukawa, donde $\lambda$ que es la constante acoplamiento de auto-interacción de los campos escalares, $m$ la masa del bosón escalar y $g$ que es el parámetro que cuantifica la intensidad de la interacción entre el campo escalar; donde se busca reproducir la curvatura de la linea de transición de fase reportada por Lattice QCD \cite{Ayala:2017_QCD}, mediante técnicas de optimización de parámetros, en este caso \textit{differential evolution}, en el lenguaje de Python y tener un conjunto amplio de combinaciones de estos parámetros que satisfacen dichas condiciones.
